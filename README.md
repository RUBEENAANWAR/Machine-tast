# HealthSync | Premium B2B Healthcare SaaS Dashboard

HealthSync is a modern, high-performance, and high-quality B2B Healthcare SaaS platform designed for healthcare professionals to manage patients, track analytics, and stay updated with real-time critical alerts. Built using contemporary design principles like glassmorphism and a rich color palette, HealthSync provides a seamless, premium user experience.

---

## ✨ Key Features

- **📊 Comprehensive Dashboard**: Real-time overview of patient admissions, active cases, and revenue trends with interactive charts.
- **🏥 Patient Management**: Advanced grid and list views for efficient patient tracking, filtering, and detailed profiles.
- **📈 Advanced Analytics**: Deep dive into patient statistics and performance metrics powered by Recharts.
- **🚨 Smart Notifications**: Seamlessly integrated critical patient alerts in the sidebar, ensuring immediate attention for emergency situations.
- **📱 Responsive Layout**: Fully responsive glassmorphism UI that adapts perfectly to different screen sizes.
- **🎨 Modern UX/UI**: Vibrant color schemes, glassmorphism cards, and smooth animations using Inter typography.

## 🚀 Technology Stack

- **Framework**: [React.js](https://reactjs.org/) (v18+)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: Vanilla CSS with Modern Tokens
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Types**: [TypeScript](https://www.typescriptlang.org/) for type safety

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Recommended: Lts/Hydrogen)
- [npm](https://www.npmjs.com/) (bundled with Node.js)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/RUBEENAANWAR/Machine-tast.git
   cd Machine-tast
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.
